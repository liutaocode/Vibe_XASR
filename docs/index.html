    本地 · 隐私 · 极速 · 中英文自由说

    一切都在你的设备上发生——隐私不上云，极速上屏，中英文随心切换。

    100% 本地，隐私至上

    • 全程离线在设备上运行，语音和文字绝不上云
    • 本地历史可查看、逐条复制 / 编辑 / 导出（JSON / 纯文本）
    • 关闭历史也有 60 秒缓冲（带倒计时），不怕说半天凭空消失
    • 使用统计：累计已说多少字、为你节省多少时间

    又快又准（X-ASR 驱动）

    • 流式 X-ASR (zh-en) 识别引擎 + FireRedVAD 端点检测
    • 中英混说无缝切换（code-switching），中文自动去字间空格
    • 延迟档位自选 160 / 480 / 960 / 1920ms，模型按需下载
    • VAD 引擎可切换（FireRed / silero）

    原生体验

    • 纯 Swift 原生 Universal 应用(Apple Silicon + Intel),菜单栏常驻、Dock 可显隐
    • 可自定义热键、首次启动引导（先体验后授权）
    • 内置便笺、可选"每次说话覆盖剪贴板"
    • 界面支持 中 / 英 / 日 / 韩，自动跟随系统、随时切换

    适合这些场景

    • Vibe Coding —— 边说边写，让灵感不断流
    • 会议记录 / 写作表达，长文不再手酸
    • 聊天、邮件、文档输入，落到任意光标处
    • OnCall 常驻待命，永远为你随时听写

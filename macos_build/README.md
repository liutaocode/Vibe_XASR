# Vibe XASR — Local Voice Input (macOS)

Hold a hotkey, speak, and the text lands right at your cursor.
**100% local and offline — your voice and text never leave your device.**

Universal build (Apple Silicon + Intel), signed and notarized by Apple.

## Download

Get the latest signed & notarized build from
**[Releases](https://github.com/liutaocode/Vibe_XASR/releases/latest)**.

Open the `.dmg`, drag **Vibe XASR** into Applications, and launch it.
On first launch, grant Microphone, Accessibility, and Input Monitoring when prompted.

## Features

- **Three dictation modes** — insert-on-finish, streaming, and OnCall stand-by
- **Push-to-talk** — hold a global hotkey to dictate, release to insert
- **Built-in notes & history** — keep, edit, and export your text
- **Localized UI** — Chinese / English / Japanese / Korean
- **Auto-update** — new versions install in place

## Requirements

macOS 15 (Sequoia) or later.

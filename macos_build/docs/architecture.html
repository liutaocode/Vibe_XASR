要点:
· 全程本地:音频与文本只在本机处理与存储,任何环节不联网(模型下载除外)。
· 实时:流式分块推理,单块计算小,Apple Silicon CPU 实时因子 ≪ 1,无需 GPU/ANE。
· 三种模式共用同一引擎,仅"何时分句 + 如何落地文本"不同。
· 权限:麦克风(采集)、辅助功能(把文本/键事件送进其它 app)、输入监听(全局热键)。
